DataFrames

We introduce the pandas DataFrame as the central structure for tabular data, and cover how to create, inspect, and perform basic operations on one.

import pandas as pd

# Define dictionary with data
sales = {
    "product":  ["coffee", "tea", "juice", "sandwich", "cookie"],
    "price":    [29.90, 35.00, 12.50, 48.00, 22.75],
    "quantity": [3, 5, 0, 2, 8]
}

# Create a small DataFrame from a dictionary
df = pd.DataFrame(sales)
print(df)
print(df.head(3))
print(df.tail(2))

# info() gives a summary of the DataFrame
df.info()

# describe() gives summary statistics for numeric columns
print(df.describe())

# Select a single column
print(df["price"])

# Apply functions on column
print(df["price"].mean())    # average price
print(df["price"].sum())     # total
print(df["price"].max())     # maximum

# Indicate if product is on sale
df["on_sale"] = [True, True, False, True, False]

# Calculate total revenue for each product
df["revenue"] = df["price"] * df["quantity"]
print(df)

Single brackets return a Series — a single column with its index on the left and no column header. Double brackets return a DataFrame — notice the column header and the slightly different layout. A Series is to a DataFrame what a single column is to a spreadsheet. You can only select multiple columns with double brackets since you need to pass a list of names. Trying df["name", "grade"] will raise a KeyError.

Exercise 1.10.5

Create a DataFrame from the weather observations table below.

city temp_celsius humidity cloudy
Oslo 12.3 72 True
Bergen 8.7 85 True
Trondheim 15.1 60 False
Stavanger 10.5 78 True
Tromsø 3.2 65 False

Then add the following three new columns and print the final result:

  1. temp_fahrenheit — the temperature converted from Celsius to Fahrenheit using the formula: (temp_celsius * 9/5) + 32. Round to one decimal place.
  2. humidity_fraction — the humidity converted from a percentage to a fraction by dividing by 100.
  3. heat_index — a simplified heat index computed as: temp_celsius + 0.33 * humidity - 4. Round to one decimal place.
Solution 
weather = {
    "city":         ["Oslo", "Bergen", "Trondheim", "Stavanger", "Tromsø"],
    "temp_celsius": [12.3, 8.7, 15.1, 10.5, 3.2],
    "humidity":     [72, 85, 60, 78, 65],
    "cloudy":       [True, True, False, True, False],
}

df_weather = pd.DataFrame(weather)

# 1. Temperature in Fahrenheit
df_weather["temp_fahrenheit"] = ((df_weather["temp_celsius"] * 9/5) + 32).round(1)

# 2. Humidity as a fraction
df_weather["humidity_fraction"] = df_weather["humidity"] / 100

# 3. Simplified heat index
df_weather["heat_index"] = (df_weather["temp_celsius"] + 0.33 * df_weather["humidity"] - 4).round(1)

print(df_weather)

Each new column is computed from one or more existing columns using element-wise operations. Notice that heat_index combines two columns — temp_celsius and humidity — in a single formula, which is applied to every row simultaneously.
