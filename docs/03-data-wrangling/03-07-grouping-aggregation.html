Grouping and aggregation

We introduce grouping as a way to split a DataFrame into subgroups and compute summary statistics for each group. We also cover how to compute multiple aggregations in a single step, and using group statistics to create new columns without collapsing the original data.

import pandas as pd

titanic = pd.read_csv("data/titanic.csv")

# ── The basic groupby ────────────────────────────────────────

# Create grouped object
group_by_sex = titanic.groupby("sex")

# Survival rate by sex
group_by_sex["survived"].mean()

# Count 1st, 2nd, 3rd class by sex
group_by_sex["pclass"].value_counts()

# ── Grouping by multiple columns ─────────────────────────────

# Survival rate by class AND sex
titanic.groupby(["sex", "pclass"])["survived"].mean()

# ── Multiple aggregations with .agg() ────────────────────────

titanic.groupby("pclass").agg(
    survival_rate = ("survived",  "mean"),
    mean_age      = ("age",       "mean"),
    max_fare      = ("fare",      "max")
    )

# ── transform() — group statistics without collapsing rows ──

# Add a column showing the mean fare for each passenger's class
titanic["class_mean_fare"] = titanic.groupby("pclass")["fare"].transform("mean")
titanic[["pclass", "fare", "class_mean_fare"]].head(10)

# How much more or less did each passenger pay compared to their class average?
titanic["fare_vs_class_avg"] = (titanic["fare"] - titanic["class_mean_fare"]).round(2)
titanic[["pclass", "fare", "class_mean_fare", "fare_vs_class_avg"]].head()

Exercise 3.7.2

The flights dataset records monthly passenger numbers for US airlines from 1949 to 1960. This exercise shows why groupby is more convenient than a for loop for computing group summaries.

import pandas as pd

flights.head()

import pandas as pd

flights = pd.read_csv("data/flights.csv")

The groupby version achieves the same result in a single line. The for loop requires filtering the DataFrame manually for each year, computing the sum, and formatting the output — three steps that groupby handles automatically. This is the core motivation for groupby: anything you can do with a for loop over groups, you can do more concisely and efficiently with groupby.

    • max_spending: maximum of Spending_USD

Task 2: .transform() returns a Series with the same number of rows as the original DataFrame — each row receives the mean life expectancy for its country. This is the key difference: .agg() collapses the data to one row per group, while .transform() broadcasts the group statistic back to every row in the group, keeping the original shape intact.

Task 4: The rows below the long-run average tend to cluster in the earlier years of the dataset for each country — reflecting the fact that life expectancy has generally increased over time, so earlier years naturally fall below the long-run mean. Any sharp dips in specific years may reflect historical events such as disease outbreaks or periods of economic hardship.

Exercise 3.7.5

In exercise 3.6.5 you decided to fill missing age values in the Titanic dataset with the overall median age. A better approach is to fill with the median age within each passenger class — since first, second, and third class passengers likely had different age profiles.

import pandas as pd

titanic = pd.read_csv("data/titanic.csv")
print(titanic.groupby("pclass")["age"].median())
  1. How different are the median ages across the three passenger classes? Does it seem worth using class-specific medians rather than the overall median?
  2. Use .transform() to create a column age_median_by_class containing the median age for each passenger’s class.
  3. Use this column to fill the missing values in the age column. Assign the result back to titanic["age"] and verify that no missing values remain.
  4. Compare the class-specific medians used here against the overall median from exercise 3.6.5. For which passenger class would the overall median have been the most misleading imputation?
Solution 
import pandas as pd

titanic = pd.read_csv("data/titanic.csv")

# 1. Median age by class
print(titanic.groupby("pclass")["age"].median())

# 2. Add median age by class as a new column
titanic["age_median_by_class"] = titanic.groupby("pclass")["age"].transform("median")
# 3. Fill missing age with class-specific median
titanic["age"] = titanic["age"].fillna(titanic["age_median_by_class"])
print(titanic["age"].isna().sum())

Task 1: The median ages differ noticeably across classes — first class passengers tend to be older than third class passengers. This makes intuitive sense: older, more established individuals were more likely to afford a first class ticket. Using class-specific medians is clearly preferable to a single overall median.

Task 3: After filling, there are zero missing values in age.

Task 4: The overall median would have been most misleading for first class passengers, whose median age is notably higher than the overall median. Imputing a younger age for older first class passengers would introduce a systematic bias into any analysis involving age.
