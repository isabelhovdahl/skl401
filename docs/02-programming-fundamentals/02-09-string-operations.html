String operations

We introduce the most useful string methods for data work and show how the same methods are available on pandas DataFrame columns through the .str accessor.

# ── Strings are sequences ────────────────────────────────────

country = "Norway"

print(len(country))     # 6
print(country[0])       # N
print(country[-1])      # y
print(country[0:3])     # Nor

# ── Essential string methods ─────────────────────────────────

name = "  Alice Andersen  "

print(name.upper())         # "  ALICE ANDERSEN  "
print(name.lower())         # "  alice andersen  "
print(name.title())         # "  Alice Andersen  " (capitalise each word)
print(name.strip())         # "Alice Andersen" (removes leading/trailing whitespace)

# ── replace and split ────────────────────────────────────────

product = "Green Tea, Organic"

print(product.replace("Green", "Black"))   # "Black Tea, Organic"
print(product.split(","))                  # ['Green Tea', ' Organic']

# ── Checking string content ──────────────────────────────────

email = "student@nhh.no"

print(email.startswith("student"))    # True
print(email.endswith(".com"))         # False
print("@" in email)                   # True

# ── f-strings for building text ──────────────────────────────

country    = "Norway"
population = 5_400_000
gdp        = 82500.756

summary = f"{country} has a population of {population:,} and a GDP per capita of ${gdp:.2f}."
print(summary)

# ── Combining strings with conditionals and loops ────────────

# Clean a messy list of country names
raw_names = ["  norway", "DENMARK ", "sweden  ", " Finland"]

cleaned = []
for name in raw_names:
    cleaned.append(name.strip().title())

print(cleaned)

# ── The .str accessor in pandas — a preview ──────────────────

import pandas as pd

df = pd.DataFrame({"country": ["  norway", "DENMARK ", "sweden  ", " Finland"]})
print(df)

df["country_clean"] = df["country"].str.strip().str.title()
print(df)

Exercises

Exercise 2.9.1

The code below applies several string methods to a variable. Without running it, predict whether city is modified after each line. Then run the code to check your answers. Finally, rewrite the code so that the cleaned result is stored in a new variable called city_clean and print it.

city = "  oslo  "

city.strip()
city.upper()
city.replace("o", "0")

print(city)
Solution

city is not modified by any of the method calls — strings are immutable, so every method returns a new string without changing the original. The final print(city) still prints " oslo ". To store the results, you must assign them back to a variable:

city = "  oslo  "

city_clean = city.strip().upper()
print(city_clean)
print(city)
OSLO
  oslo
Notice that methods can be chained: .strip() removes the surrounding spaces first, and .upper() is then applied to the stripped result. The original city is still unchanged. This is a common source of confusion — always remember to assign the result of a string method call if you want to keep it.

Exercise 2.9.2

Use string methods combined with conditionals to process the list of filenames below. For each filename:

  1. Print a message if it ends with ".csv".
  2. Print a message if it starts with "raw_".
  3. Print a message if it both starts with "raw_" and ends with ".csv".
filenames = [
    "raw_data.csv",
    "raw_data.xlsx",
    "cleaned_data.csv",
    "report.pdf",
    "raw_survey.csv",
]
Solution 
filenames = [
    "raw_data.csv",
    "raw_data.xlsx",
    "cleaned_data.csv",
    "report.pdf",
    "raw_survey.csv",
]

for filename in filenames:
    if filename.endswith(".csv"):
        print(f"{filename} is a CSV file.")
    if filename.startswith("raw_"):
        print(f"{filename} is a raw data file.")
    if filename.startswith("raw_") and filename.endswith(".csv"):
        print(f"{filename} is a raw CSV file and should be cleaned before use.")
raw_data.csv is a CSV file.
raw_data.csv is a raw data file.
raw_data.csv is a raw CSV file and should be cleaned before use.
raw_data.xlsx is a raw data file.
cleaned_data.csv is a CSV file.
raw_survey.csv is a CSV file.
raw_survey.csv is a raw data file.
raw_survey.csv is a raw CSV file and should be cleaned before use.
.startswith() and .endswith() return boolean values, making them natural companions to if statements. Notice that these are three separate if statements rather than if/elif — a file can satisfy more than one condition simultaneously, so all three need to be evaluated independently.

Exercise 2.9.3

The list below contains a mix of values stored as strings — some are whole numbers and some are not. Use a loop with string methods and conditionals to build two new lists: one containing the numeric values converted to integers, and one containing the non-numeric values left as strings. Print both lists when the loop is finished.

raw_values = ["12", "oslo", "8", "3", "bergen", "42", "7", "trondheim", "5"]
Hint

.isdigit() is a string method that returns True if all characters in the string are digits and False otherwise. For example, "42".isdigit() returns True while "oslo".isdigit() returns False.

Solution 
raw_values = ["12", "oslo", "8", "3", "bergen", "42", "7", "trondheim", "5"]

numeric     = []
non_numeric = []

for value in raw_values:
    if value.isdigit():
        numeric.append(int(value))
    else:
        non_numeric.append(value)

print(f"Numeric values: {numeric}")
print(f"Non-numeric values: {non_numeric}")
Numeric values: [12, 8, 3, 42, 7, 5]
Non-numeric values: ['oslo', 'bergen', 'trondheim']
.isdigit() returns True if all characters in the string are digits, making it a clean and simple way to check whether a string can safely be converted to an integer. Note that this only works for whole numbers — "12.5".isdigit() returns False because the decimal point is not a digit.

Exercise 2.9.4

The string below contains a comma-separated list of country names with inconsistent formatting — extra spaces and mixed capitalisation. Split it into individual country names, clean each one so that it has no leading or trailing spaces and starts with a capital letter, and then reassemble the cleaned names into a single string separated by " | ". Print the final result.

countries = "norway ,  GERMANY, brazil ,france,   JAPAN  "
Hint

To reassemble the cleaned names, you can either use a for loop or the string method join().

Solution 
countries = "norway ,  GERMANY, brazil ,france,   JAPAN  "

# Split into individual names
parts = countries.split(",")

# Clean each name
cleaned = [part.strip().capitalize() for part in parts]

# Approach 1: reassemble using a for loop
result_loop = ""
for i in range(len(cleaned)):
    if i == 0:
        result_loop += cleaned[i]
    else:
        result_loop += " | " + cleaned[i]
print(result_loop)

# Approach 2: reassemble using .join()
result_join = " | ".join(cleaned)
print(result_join)
Norway | Germany | Brazil | France | Japan
Norway | Germany | Brazil | France | Japan
.split(",") breaks the string into a list at every comma. .strip() removes surrounding spaces from each part and .capitalize() ensures the first letter is uppercase and the rest are lowercase. For reassembling, .join() is the most concise approach — it takes a list of strings and joins them into a single string using the specified separator. The for loop approach makes the logic more explicit: the first element is added without a separator, and every subsequent element is prefixed with " | ". Both approaches produce the same result.

Exercise 2.9.5

Write a function called clean_string that takes a messy string as input and returns a cleaned version by:

  1. Removing leading and trailing spaces.
  2. Converting to lowercase.
  3. Replacing any internal spaces with underscores.

Then apply it to the list of column names below using a list comprehension and print the result.

column_names = ["  GDP per Capita  ", "Country Name", " Population  ", "CONTINENT", "  Life Expectancy  "]
Solution 
def clean_string(s):
    return s.strip().lower().replace(" ", "_")

column_names = ["  GDP per Capita  ", "Country Name", " Population  ", "CONTINENT", "  Life Expectancy  "]

cleaned_names = [clean_string(name) for name in column_names]
print(cleaned_names)
['gdp_per_capita', 'country_name', 'population', 'continent', 'life_expectancy']
The three methods are chained in order: .strip() removes surrounding spaces first, .lower() converts to lowercase, and .replace(" ", "_") handles any remaining internal spaces. The order matters — if .replace() were applied before .strip(), leading and trailing spaces would be converted to underscores rather than removed. Wrapping the cleaning logic in a function makes it easy to apply consistently to any list of strings — a very common task when preparing data for analysis.
